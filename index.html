什么是Redis？​ Redis是一种基于键值对（key-value）的NoSql数据库。它不是一般的键值对数据库，它比普通的键值对强大的地方是，Redis中的value支持string（字符串）、hash（哈希）、 list（列表）、set（集合）、zset（有序集合）、Bitmaps（位图）、 HyperLogLog、GEO（地理信息定位）等多种数据结构，因此Redis可以适用很多应用场景。 ​ Redis将所有的数据都放在内存中，所以Redis的读写性能很出色。不仅如此，Redis还可以将内存的数据，利用快照和日志的形式保存到硬盘上，这样可以确保以后机器故障之后，数据不会‘’丢失“。 ​ 除了上述功能以外，Redis还提供了键过期、发布订阅、事务、流水线、Lua脚本等附加功能。 ​ 总的来说，Redis是一款性能强大的开源工具。 Redis有哪些数据结构？ String 常见命令 set key value（添加一个键值对） get key（根据键获取对应的值） del key（根据key删除对应的键值对） mset key1 value1 key2 value2 ...
